# movie

## 项目简介

项目简介:`movie`即刻影院-在线电影票预订系统,一个基于`Spring Boot + Mybatis`的简单项目，包含用户端和管理端，
主要功能包括用户个人资料与密码的修改，头像的上传，电影票浏览与购买，电影信息管理，用户订单管理等，为Java课程设计的作业。
